# 1. Base de Datos Relacional
Sistema que guarda datos en tablas relacionadas entre sí mediante claves.

# 2. Tabla
Estructura que organiza datos en filas y columnas.

# 3. Relación
Vínculo entre tablas usando claves primarias y foráneas.

# 4. Columna
Representa un atributo o campo del dato (ej. nombre, edad).

# 5. Renglón (fila)
Representa un registro único de información.

# 6. Motor de Base de Datos Relacional
Software que crea, administra y consulta bases de datos relacionales.

# 7. Motores más populares
MySQL, PostgreSQL, Oracle, SQL Server, MariaDB.

# 8. Motor ligero sin servidor
SQLite.

# 9. SQL
Lenguaje para crear, consultar y modificar bases de datos relacionales.

# 10. Elementos de un Sistema de Información Relacional
Base de datos, motor de BD, SQL, usuarios, aplicaciones.
